# AI Customer Support Bot

Handles inbound support messages with Claude AI, fetches customer history from Zoho CRM, generates contextual responses, detects escalation triggers like refund or legal requests, and routes to a human agent when needed.

## Quick Start
1. Download `workflow.json`
2. Import into n8n: **Settings → Import Workflow**
3. Replace `${CLIENT_ID}` and `${CLIENT_NAME}` with your values
4. Configure credentials (Zoho CRM, Claude API)
5. Activate and test
